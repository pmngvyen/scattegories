One-time setup

This page needs a Firebase Realtime Database to pass moves between players. It is free and takes a few minutes.

1. Go to console.firebase.google.com and create a project.
2. Databases & Storage → Realtime Database → Create database. Pick a region, choose test mode.
3. Open the Rules tab straight away and publish the rules below — test mode expires after about a month and then blocks everything.
4. Gear icon → Project settings → Your apps → Web (</>) → register, then copy the config values.
5. Paste them into the FIREBASE_CONFIG block near the top of this file's script.

If databaseURL is missing from the snippet, copy it from the top of the Realtime Database Data tab. Note this is Realtime Database, not Firestore. 

{
  "rules": {
    "rooms": {
      "$code": { ".read": true, ".write": true }
    }
  }
}
